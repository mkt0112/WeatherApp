WeatherApp

WeatherApp is a simple Android application that allows users to search for weather conditions in different cities. It fetches weather data from the OpenWeatherMap API and displays it to the user. Additionally, it provides functionality to retrieve historical temperature data from a Firebase database.

Features

- Search City: Users can search for weather conditions in different cities using the search bar.
- Current Weather:Displays the current temperature, maximum temperature, minimum temperature, weather condition, and city name.
- Historical Data:Users can retrieve historical temperature data for specific dates from a Firebase database.

Getting Started

To get started with WeatherApp, follow these steps:

1. Clone this repository to your local machine.
2. Open the project in Android Studio.
3. Build and run the project on an Android device or emulator.

Usage

- Search for Weather: Enter the name of a city in the search bar and press Enter to search for weather conditions in that city.
- Retrieve Historical Data: Enter a date in the input field and press the "Retrieve" button to retrieve historical temperature data for that date.

Dependencies

WeatherApp uses the following dependencies:

- Retrofit: For making network requests and handling API responses.
- Gson: For converting JSON responses to Kotlin objects.
- Firebase Realtime Database: For storing and retrieving historical temperature data.
